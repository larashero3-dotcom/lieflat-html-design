结论

按工作距离路由。

短答案走便宜路线。长任务走聪明路线。不可逆任务必须加门。

A

Instant / low：高频分类、改写、抽取。

B

Default / medium：日常编码和成本敏感助手回合。

C

GPT-5.5：复杂编码、研究、检索和多工具流程。

D

Pro / xhigh：少量、深度、慢速但高价值决策。

主要发现

不是更大的聊天框，而是更小的管理负担。

GPT-5.5 的产品问题不是“它能不能回答”，而是“它能不能在你少操心的情况下，把复杂工作推进到可验收状态”。
