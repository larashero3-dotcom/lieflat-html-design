# Asset Licenses

This repository bundles HTML/CSS/SVG/canvas/WebGL templates. The `story-field` template also bundles a small Asterion sample image pack so the image-led template has a real cinematic preview.

## Template Assets

- Template HTML/CSS/JS: MIT, unless otherwise noted.
- Generated CSS/SVG/canvas/WebGL visuals inside templates: MIT, unless otherwise noted.
- Placeholder geometry and abstract decorative elements: MIT.

## Story Field Sample Assets

- `skills/lieflat-html-deck/assets/templates/story-field/asterion_assets/*.jpg`: NASA-origin space imagery used as sample imagery for the Asterion/Story Field template. Do not imply NASA endorsement.
- Story Field previously used bundled KTF Metro font files. Those font files are not included in this repository; the public template uses system fonts plus Google Fonts fallbacks instead.

## External Dependencies

Some templates may reference Google Fonts via remote CSS. If you need fully offline or single-file distribution, replace these font imports with system font stacks or bundled fonts before publishing a final deck.

## User-Provided Images

Images added to generated decks are not covered by this repository license. Record image provenance in `images/manifest.json` when producing a deck.
